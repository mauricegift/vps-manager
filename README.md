# VPS Manager

Professional full-stack VPS control panel built by **[Gifted Tech](https://me.giftedtech.co.ke)**.

Manage PM2 processes, Docker containers, databases, files, and SSH terminals across unlimited remote servers.

## Features
- SSH terminal with dark/light themes
- PM2 process manager with version badges
- Docker container & image manager
- PostgreSQL & MySQL browser
- File manager with editor
- Remote VPS multi-server support
- Extras / software manager

## Quick Start
```bash
bash install.sh
npm start
```
